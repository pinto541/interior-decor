# INTERIOR DECOR Website

Premium Vite + React website for INTERIOR DECOR, Dimapur, Nagaland.

## Installation

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
npm run preview
```

## Tech Stack

- Vite
- ReactJS
- JavaScript
- Tailwind CSS
- React Router DOM
- Framer Motion
- React Icons
- SwiperJS

## Project Structure

```text
src/
├── assets/
│   ├── images/
│   └── logo/
├── components/
├── data/
├── pages/
├── App.jsx
├── main.jsx
└── index.css
```

## Image Notes

Portfolio and service images currently use premium placeholder image URLs. Replace them in:

- `src/data/portfolio.js`
- `src/data/services.js`

Add final brand assets to:

- `src/assets/images/`
- `src/assets/logo/`
